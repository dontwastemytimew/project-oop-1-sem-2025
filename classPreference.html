Detailed Description

Клас моделі даних для збереження критеріїв пошуку (вподобань).

Цей клас інкапсулює набір фільтрів, які користувач встановлює на сторінці пошуку для знаходження потенційних партнерів. Об'єкт цього класу передається в DatabaseManager для формування SQL-запитів фільтрації.

Основні параметри:

  • Віковий діапазон.
  • Географія (місто).
  • Гендерна ідентичність та орієнтація.
