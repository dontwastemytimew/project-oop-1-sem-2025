Detailed Description

Віджет сторінки "Мій Профіль" (Редагування).

Цей клас надає інтерфейс для створення нового або редагування існуючого профілю користувача. Він містить форму з полями для введення особистих даних (ім'я, вік, місто, біографія, теги) та функціонал для завантаження фото.

Основні можливості:

  • Валідація вводу (обов'язкові поля, перевірка віку).
  • Автодоповнення для поля "Місто".
  • Вибір фотографії з файлової системи.
  • Збереження або оновлення даних у базі через DatabaseManager.
