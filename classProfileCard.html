Detailed Description

Віджет "Картка Профілю".

Цей клас відповідає за візуальне відображення інформації про користувача у вигляді картки (схоже на Tinder). Він компонує фото, ім'я, вік, місто, біографію та теги в єдиний UI-елемент.

Використовується переважно на сторінці пошуку (SearchPageWidget) для показу кандидатів на метч.

void ProfileCard::setProfileData ( const UserProfile &  profile)

Заповнює картку даними з об'єкта UserProfile.

Метод оновлює всі текстові поля (ім'я, вік, місто, біо) та завантажує фотографію профілю. Якщо фото відсутнє, встановлюється заглушка (placeholder). Також форматує та відображає список тегів.
