Detailed Description

Структура для представлення одного повідомлення в чаті.

Ця структура використовується як DTO (Data Transfer Object) для передачі даних повідомлення між базою даних, логікою (ChatManager) та інтерфейсом (ChatPageWidget). Вона містить всю необхідну інформацію для відображення одного "бабла" в історії листування.
